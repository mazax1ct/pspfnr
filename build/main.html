Юридическая поддержка

Бесплатные консультации собственного департамента юристов Профсоюза для членов организации. Ответим на вопросы, поможем с документами, сопроводим в непредвиденных жизненных ситуациях. Только для участников ПСПФНР.

Консультируем во всех областях права.

Горячая линия юридической поддержки:

8-800-350-33-85

Звонок бесплатный ПН-ПТ: 9.00-18.00 СБ-ВС: Выходной
